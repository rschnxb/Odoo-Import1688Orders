1688采购订单导入

快速从1688导入采购订单到Odoo

功能特性

✅ 快速导入

从1688导出的Excel文件一键导入到Odoo采购订单

🏢 自动创建供应商

自动识别并创建或匹配供应商信息

📦 自动创建产品

自动识别并创建或匹配产品信息，支持产品编码和SKU匹配

📊 详细报告

提供详细的导入摘要报告，包括成功、失败和跳过的订单

使用方法

  1. 从1688下载采购订单Excel文件
  2. 在Odoo中进入：采购 → 导入1688订单
  3. 上传Excel文件
  4. 点击"开始导入"按钮
  5. 查看导入摘要和新创建的采购订单

安装要求

本模块需要安装Python库：openpyxl

pip install openpyxl
